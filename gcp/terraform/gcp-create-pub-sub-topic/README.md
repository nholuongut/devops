### Features
  
- Create a Pub/Sub topic

Create a Pub/Sub topic.
-------------

**Script Name:** create-pub-sub.tf

**Properties File:** variables.tf

**Usage:** terraform `<operation>`

***Where,***
```
operation=init OR
operation=plan OR
operation=apply OR
operation=destroy
```

**e.g.**
To create a Pub/Sub Topic
`terraform init`
`terraform plan`
`terraform apply`

To delete the Topic
`terraform destroy`
